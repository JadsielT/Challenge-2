# Challenge-2
Este repositorio contiene la libreta de jupyter referente al proyecto Challenge_1_Data science de Alura; El cual pretende servir de practica para el procesamiento  analisis de datos usando la libreria pandas 
el objetivo principal es analizar los datos de 4 tiendas para poder concluir cual es la que se va a cerrar para poder usar el dinero en otra inversion 

En este proyectosse utilizaron las librerias:

-Pandas
-matplotlib.pyplot
 Para el manejo de los datos y para la graficacion de los datos a mostar 


 Se crearon 5 etapas las cuales corresponden a los ejes que se deben de analizar segun lo establecido en el challenge 
 **Es importante ejecutar en orden las celdas de codigo, de otroa forma puede cambiar el resultado o levantar algun error**
